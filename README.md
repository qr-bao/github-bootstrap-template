# GitHub Bootstrap Template

This folder is the starting point for a computer that should configure itself
from GitHub and Markdown.

## Read Order

1. `README.md`
2. `INSTRUCTIONS.md`
3. `RULES.md`
4. `MACHINE.md`
5. `STATUS.md`

## File Roles

- `README.md` = entry point and map
- `INSTRUCTIONS.md` = how AI should start and set things up
- `RULES.md` = permanent behavior and safety rules
- `MACHINE.md` = computer-specific values
- `STATUS.md` = current state and last sync result

## Use

Copy this folder into a GitHub repo, then let AI read `README.md` first.
